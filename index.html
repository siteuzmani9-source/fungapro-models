Privacy Policy – Funga Pro

Last updated: December 2025

Funga Pro (“we”, “our”, “the app”) respects your privacy. This Privacy Policy explains how information is collected, used, and protected when you use the Funga Pro mobile application.

1. Purpose of the App

Funga Pro is an educational application that analyzes mushroom images using artificial intelligence. The app does not provide medical advice.

2. Information We Collect

a) Photos

  • Photos are taken only when the user chooses to scan
  • Images are sent to an AI service for analysis
  • Photos are not permanently stored

b) Location (Optional)

  • Collected only with user permission
  • Used to show nearby mushroom scans
  • Not shared with third parties

c) Scan History

  • Scan results, confidence score, timestamp
  • Stored securely using Firebase

3. Data Usage

Data is used only for app functionality and educational purposes.

4. Data Sharing

We do not sell or share user data with third parties.

5. Data Security

Data is protected using Firebase security mechanisms.

6. Children’s Privacy

This app is not designed for children under 13.

7. Disclaimer

Results are not guaranteed to be accurate. Never consume mushrooms based on this app.
