Last updated: December 2025 Funga Pro (“we”, “our”, “the app”) respects your privacy. This Privacy Policy explains how information is collected, used, and protected when you use the Funga Pro mobile application. 1. Purpose of the App Funga Pro is an educational application that analyzes mushroom images using artificial intelligence. The app does not provide medical, health, or safety advice. 2. Information We Collect a) Photos - Photos are taken only when the user explicitly chooses to scan - Images are sent temporarily to an AI service for analysis - Photos are not permanently stored on our servers b) Location (Optional) - Location data is collected only with user permission - Used solely to display nearby mushroom scan locations on the map - Location data is not shared with third parties c) Scan History - Scan result (predicted class) - Confidence score - Timestamp - Optional location data (if permission granted) - Stored securely using Firebase services 3. Data Usage Collected data is used only for application functionality and educational purposes. 4. Data Sharing We do not sell, rent, or share user data with third parties. Third-party services used: - Firebase (data storage and security) - AI inference service (image analysis only) 5. Data Retention Scan history data is stored only as long as necessary to provide app functionality. Users may uninstall the app at any time to stop all data collection. 6. Data Security We use Firebase security mechanisms and industry-standard protections to safeguard user data. 7. Children’s Privacy Funga Pro is not designed for children under the age of 13 and does not knowingly collect personal data from children. 8.
